# Bubble sort
버블 정렬은 단순하다 배열 첫번째 요소 부터 다음 요소를 비교해서 큰 값을 뒤로 보낸다.  
그리고 나서 순환 수를 한번씩 줄여서 그 작업을 반복한다.  
비용이 많이 드는 연산이지만 복잡하지 않다.

![wikipedia_bubble_sort](../../img/bubble_sort.gif)
